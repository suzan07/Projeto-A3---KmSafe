# Projeto-A3---KmSafe
Aplicativo de mobilidade urbana inspirado na Uber, com validação rigorosa de motoristas por meio de análise de documentos, antecedentes e reconhecimento facial. O motorista terá até 2 minutos para cancelar a corrida após a aceitação, garantindo mais segurança e compromisso com os passageiros.
